## Overview
For those unable to download the app, I've used this space to showcase the main features and user interfaces I've developed. <br/><br/>
The app allows users to effortlessly find and filter in-home services by location or type, access detailed information with images, pricing, business hours and reviews, and save favorite services. Users can keep track of their orders, communicate with service providers through secure in-app messaging, and customize the settings for a personalized, secure, and efficient experience. <br/><br/>
The app prioritizes user safety by verifying each service provider, ensuring a secure and trustworthy experience. Utilized reliable third-party tools like Stripe for seamless and secure payment processing. <br/><br/>
The App incorporates a variety of interesting tools and frameworks, and I would be delighted to discuss the frameworks that I'm proficient in with you! <br/><br/>

## KeyWords
* SwiftUI.
* Multithreading.
* MVVM Architecture.
* RESTful API Integration.
* Push Notifications.
* In-App Messaging.
* User Authentication: Phone number Verification, Email/Password verification, Google sign in, Apple sign in.
* Firebase: DataBase, CloudStorage, Security Rules, Cloud Functions, Analytics.
* Stripe: Payment Processing, Receipt Handling, Tax/Invoice Creating, Transfer & Payout.
* SendGrid: Transactional Emails.
